# Express Backend for Nuxt Recipe App

This backend will:
- Expose REST API endpoints for recipes and users
- Connect to a PostgreSQL database
- Handle authentication and business logic

## Getting Started

1. Install dependencies:
   ```bash
   cd backend
   npm install
   ```
2. Set up your database (PostgreSQL recommended)
3. Configure environment variables in a `.env` file
4. Run the server:
   ```bash
   npm run dev
   ```

---

You will learn:
- How to build REST APIs with Express
- How to connect to a database
- How to migrate features from Supabase to your own backend
- How to secure your API with authentication
